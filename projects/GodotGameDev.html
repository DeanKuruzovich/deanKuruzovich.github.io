Game Development

I have made several partially developed games using the Godot Game Engine. I enjoy Godot because it is open source and has an active community of developers.

Arcade Chess

I partially built an arcade-style chess game intended for mobile. The goal was to survive as the game continuously spawned pieces and moved them in an attempt to checkmate you. You could also gain more pieces by capturing the computer's pieces or completing other tasks.

For this project I had to write my own chess engine to handle the game's unique nature of continuously spawning pieces. This proved to be a challenge — the engine was very slow, taking over five seconds to make a move even with a small number of pieces on the board, despite running on multiple threads on a MacBook.

Simulations

I built a number of simulations of things I found interesting. Godot worked well for this since I'm already familiar with it and it is easy to customize. It handles moderately large simulations well enough, though it uses more RAM than ideal.

Space Exploration Game
