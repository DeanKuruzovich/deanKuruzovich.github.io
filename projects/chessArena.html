Chess Arena

This is the Web game I made using HTML and Javascript.

This game I originally built in Godot, but switched to JavaScript upon relizeing the difficualtys of running Godot games on Web outweighted the effort to convert the game to a native web format.

My favorite challenge in building this game was designing the Chess engine that could work in the dynamic and fast pace conditions of the game. this was also challengeing due to the single threaded nature of Javascript makeing a engine that evaluated continuesly very difficult and impracticle.

I ended up settling on a balce of speed and preformence in the Engine that I was happy with, not making the game impossible, but making it challeging.
